## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/go-sample.git
```

2. Navigate to the project directory:
```bash
cd go-sample
```

3. Run any example:
```bash
go run basic/01_basic_syntax.go
```

## Requirements

- Go 1.16 or later
- SQLite3 (for database examples)
